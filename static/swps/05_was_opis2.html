Do analizy mediacji służy funkcja mediate z pakietu psych. Mediatory należy w deklaracji modelu umieścić w nawiasach okrągłych.

psycho_z <- psycho %>%
  select(-plec) %>% # usunięcie zmiennej płeć - nie może być standaryzowana
  rename(indec=indecisiveness) %>% # zmiana nazwy zmiennej 
  standardize() %>% # standaryzacja wszystkich cech
  mutate(plec=psycho$plec, # powrót zmiennej płeć
         plec=ifelse(test = plec == 1, yes = 1, no = -1)) # rekodowanie zmiennej płeć

model_med <- mediate(indec ~ ps + perf_cmda + (wstyd) + (poczucie_winy), data = psycho_z)

summary(model_med)

W tej analizie zostanie wykorzystany pakiet lavaan, którego funkcje i możliwości są szczegółowo opisane na stronie projektu.
